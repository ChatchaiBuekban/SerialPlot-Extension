# Change Log

All notable changes to the "serialplot-extension" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [0.0.4] - 2026-04-21

- Added **Data Logging** feature: Users can now record incoming serial data and export it as a CSV file.
- Added timestamping to the exported CSV data.

## [0.0.3] - 2026-04-21

- Added **Pause/Resume** button to freeze/unfreeze plot updates.
- Added **VS Code Theme Integration**: UI now matches the active editor theme colors and fonts.
- Improved UI styling for controls and buttons.

## [0.0.2] - 2026-04-21

- Added **Multi-Stream Support**: Plot multiple values simultaneously.
- Added **Split View**: Toggle between combined and individual charts.
- Improved rendering with **Right-Alignment** for synchronized data streams.
- Improved series initialization and robustness.
- Added Arduino code example in README.
- Fixed linting and code stability issues.

## [0.0.1] - 2026-04-21

- Initial release.
- Real-time plotting with Y-axis values.
- Implemented Smooth, Low Pass, and High Pass filtering.
- Plot capture functionality.
